## Moscow public catering market

### Описание проекта

Проведение исследовательского анализа рынка общественного питания Москвы. Исследование поможет инвесторам выбрать подходящее место для открытия заведения.

### Навыки и инструменты 

- python
- pandas
- seaborn
- matplotlib

### Общий вывод

Проведена предобработка данных (оптимизированы типы данных, удалены неявные пропуски), исследовательский анализ (распределение заведений по районам Москвы, распределение сетевых и несетевых заведений, посадочных мест, рейтингов, средний чек заведений, топ-15 популярных сетей Москвы). 

Определены наиболее популярные категории заведений, районы Москвы с самыми высокими ценами, выявлена прямая зависимость среднего рейтинга от ценовой категории заведения. Рекомендованы административные районы Москвы для открытия заведения.
