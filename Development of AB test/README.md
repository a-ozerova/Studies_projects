## Development of A/B test and analysis of the results

### Описание проекта

Нужно проверить гипотезу о том, что новый алгоритм взаимодействия с пользователями развлекательного приложения увеличит интерес пользователей.

### Навыки и инструменты 

- python
- pandas
- matplotlib
- scipy.stats
- math

### Общий вывод

Рассмотрена динамика регистраций пользователей, распределение пользователей по количеству просмотренных страниц.

Проведен расчет размера выборки и длительности теста. Проведен мониторинг А/В-теста (распределение пользователей по регионам и устройствам, наличие пересечений пользователей в группах).

Проверена гипотеза о том, что доля успешных сессий (т.е. просмотр 4 и более страниц за одну сессию) не увеличится при внедрении нового алгоритма рекомендаций, который, по мнению разработчиков, будет показывать более интересный контент для каждого пользователя. 

Гипотеза не получила подтверждение. Рекомендовано внедрить нововведение в приложение.
