## Yandex Afisha

### Описание проекта

Нужно проанализировать распределение заказов на сервисе Яндекс Афиша по сегментам и их сезонные изменения, проанализировать осеннюю активность пользователей, определить популярные события и партнеров и 
проверить выдвинутые гипотезы. 

### Навыки и инструменты 

- python
- pandas
- matplotlib
- scipy.stats

### Общий вывод

Проведена предобработка данных. Проведен статистический анализ распределения выручки в тенге и рублях и распределения количества заказов в тенге и рублях, отфильтрованы выбросы в значениях выручки.
Проанализировано распределение заказов по месяцам, сезонное (лето-осень) распределение по типам мероприятий, типам устройств, возрастному рейтингу, сезонное распределение выручки по типам мероприятий, осенняя активность пользователей, в т.ч. по дням недели.

Выявлены популярные регионы и билетные партнеры, для которых характерно наибольшее разнообразие мероприятий и наибольшее количество заказов.

Проверены гипотезы о количестве заказов и среднем времени между заказами пользователей мобильных и стационарных устройств.

Сделаны рекомендации на базе проведенного анализа.
