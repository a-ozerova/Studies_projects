## Startups research

### Описание проекта

Исследование закономерностей финансирования стартапов и оценка перспектив выхода на рынок. 

### Навыки и инструменты 

- python
- pandas
- matplotlib

### Общий вывод

Проведена предобработка данных. Отображена динамика типичного размера средств, которые стартапы получали в рамках одного раунда финансирования за 1999-2013гг. Определена доля сотрудников с образованием в компаниях и объем типичного финансирования. 

Определены категории стартапов с типично высокими ценами, стартапы с наибольшим разбросом цен за стартап и среднее число раундов финансирования для каждого статуса стартапов.

Сделаны рекомендации на основе проведенного анализа.
